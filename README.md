# generator-koa-ddd-api

Node.js koa boilerplate - REST API   

## Installation

_Requires Node 8 or greater_

```shell
npm install -g yo generator-koa-api-boilerplate
```

## Generators

```shell
yo koa-api-boilerplate # generate a new project
```
